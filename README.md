#  SQL Injection Vulnerability Scanner
A Python-based security tool designed to automate the detection of SQL Injection flaws in web applications:

## Features
- **Automated Testing:** Iterates through a custom list of SQL payloads.
- **Response Analysis:** Scans server response text for known database error signatures
- **Modular Design:** Uses external payload files for easy updates.

## Tech Stack
- Python 
- Requests Library (HTTP interaction)
